Java 重载(Overload)

日期：2024-02-17 20:43:08

java中允许同一个类中，多个重名的方法存在，但要求形参列表不一致
比如：System.out.println(); out 是 PrintStream类型

重载的好处
1.减轻了起名的麻烦
2.减轻了记名的麻烦

细节：
1.方法名：必须相同
2.形参列表：必须不同（形参类型或个数或顺序必须至少有一个不一样）
3.返回类型：无要求（跟返回类型无关）
