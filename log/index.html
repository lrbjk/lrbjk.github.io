Java 可变参数使用

可变参数
Java中允许将同一类中的多个同名同功能但参数个数不同的方法，封装成一个方法。
基本语法
访问修饰符 返回类型 方法名（数据类型… 形参名）{
}

使用细节
1.可变参数的实参可以为0个或任意多个
2.可变参数的实参可以为数组
3.可变参数的本质就是数组
4.可变参数可以和普通类型的参数一起放在形参列表，但必须保证可变参数在最后

5.一个形参列表中只能出现一个可变参数

Java 重载(Overload)

java中允许同一个类中，多个重名的方法存在，但要求形参列表不一致
比如：System.out.println(); out 是 PrintStream类型

重载的好处
1.减轻了起名的麻烦
2.减轻了记名的麻烦

细节：
1.方法名：必须相同
2.形参列表：必须不同（形参类型或个数或顺序必须至少有一个不一样）
3.返回类型：无要求（跟返回类型无关）

Java数据类型转换的细节

在Java中，有多种类型的数据混合运算时，系统首先自动将所有数据转换成容量最大的那种数据类型，然后再计算

把精度（容量）大的数据类型赋值给精度（容量）小的数据类型就会报错，反之就会进行自动类型转换。

(byte, short)和 char之间不会相互自动转换

byte，short，char 他们三者可以计算，在计算时首先转换为int类型

boolean 不参与转换

表达式结果类型自动提升为操作数中最大的类型
